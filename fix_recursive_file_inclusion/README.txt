Интересная техника устранения рекурсивного включения файлов в С/С++ коде.

Предпологается что все объявления всех символов будет размещенно не в разных хедер файлах, в все будет объявлять в одном файле
в условных заголовках. Все cpp файлы, которым нужны объявления символов должны будут подключать только этот файл со всеми объявлениями
в самом начале и опредедять ниже #define`ы нужным их секции, в которых храняться нужные объявления.
